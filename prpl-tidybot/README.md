# PRPL TidyBot

![workflow](https://github.com/tomsilver/prpl-tidybot/actions/workflows/ci.yml/badge.svg)

Code for interfacing with the real TidyBot++ robot.

## Installation

1. Recommended: create and source a virtualenv (perhaps with [uv](https://github.com/astral-sh/uv))
2. Install this repo: `pip install -e ".[develop]"`
